# Anime Splashscreens for KDE

Two splashscreens for KDE featuring popular anime girls.

- Dicta Striker from 'Umamusume Cinderella Gray'.

- Nico Yazawa from 'Love Live School Idol Project'.

- Yamada Ryo from 'Bocchi the Rock!'









# Installation

Copy the folders into  ```~/.local/share/plasma/look-and-feel```











Have fun!
